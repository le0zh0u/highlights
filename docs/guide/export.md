# 导出

## 导出到Markdown
在书本详情页中，右上角的菜单栏中找到「导出」菜单，可以选择
- 导出Markdown文件
- 复制Markdown格式

**导出Markdown文件**

点击后，指定到对应的文件夹，会生成以书名为文件名的Markdown文件。

**复制Markdown格式**

点击后，会将书本以Markdown的格式保存到剪切板中，在目标位置粘贴即可获得内容。

## 导出设置
在「偏好设置」中，找到「导出」的标签页。

### 导出Markdown

**导出Markdown中包含章节**

在导出Markdown内容时，

- 如果打开，意味着会以章节为二级标题进行导出。
- 如果打开，意味着会只导出所有划线的笔记文本。
